# 🎲 학급추첨기 (Class Lottery Picker)

교실 TV, 전자칠판, 빔프로젝터 환경에 최적화된 **학급 랜덤 추첨기**입니다.

학생 명단을 엑셀에서 그대로 복사하여 붙여넣을 수 있으며, 발표자 선정, 모둠장 선발, 청소당번 뽑기 등 다양한 학급 활동에 활용할 수 있습니다.

---

## ✨ 주요 기능

### 👨‍🎓 학생 명단 등록

* 엑셀에서 학생 명단 복사 후 붙여넣기
* 번호 + 이름 형식 자동 인식
* 중복 학생 자동 제거

예시

```text
1    김민준
2    이서연
3    박지후
4    최하린
```

---

### 🎰 랜덤 추첨

* 이름이 빠르게 변경되는 롤링 효과
* 점점 느려지며 멈추는 슬롯머신 방식
* 드럼롤 효과음 재생
* 추첨 결과 대형 표시

---

### 🎉 당첨 연출

당첨 시 다음 순서로 연출됩니다.

1. 드럼롤 효과음
2. 이름 정지
3. 화면 플래시
4. 폭죽 효과
5. 색종이 폭죽
6. 당첨 효과음 (띠링)

---

### 🏆 당첨자 관리

* 이미 뽑힌 학생은 다시 추첨되지 않음
* 당첨자 목록 자동 누적
* 남은 학생 수 표시
* "당첨자 초기화" 버튼 제공

---

### 📺 교실 TV 최적화

* 대형 글씨
* 고대비 색상
* 전체화면 모드 지원
* 교실 뒤에서도 확인 가능한 UI

---

## 🖥 화면 구성

```text
학생 명단 입력
       ↓
학생 등록
       ↓
랜덤 추첨
       ↓
드럼롤
       ↓
당첨
       ↓
폭죽 + 색종이
       ↓
당첨자 목록 저장
```

## 🚀 실행 방법

### 방법 1

1. 저장소 다운로드
2. `학급추첨기.html` 파일 실행
3. 브라우저에서 사용

### 방법 2

GitHub Pages 배포

1. 저장소 업로드
2. Settings → Pages
3. Branch: main
4. Save

배포 후

```text
https://사용자명.github.io/저장소명
```

으로 접속 가능합니다.
